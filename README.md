# Sudoku

Petit jeu de Sudoku jouable dans le navigateur (iPhone, iPad, ordinateur).

**Jouer : https://chrisblb.github.io/sudoku/**

- 4 niveaux : Facile, Moyen, Difficile, Expert
- Crayon pour les notes, sauvegarde automatique de la partie en cours
- Sur iPhone / iPad : bouton **Partager** -> **Sur l'écran d'accueil** pour l'installer comme une app

Tout se passe dans le navigateur : aucune donnée n'est envoyée, la partie est enregistrée sur l'appareil.
